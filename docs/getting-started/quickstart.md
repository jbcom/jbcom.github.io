# Quickstart

This guide will help you get started with jbcom.github.io - Jon Bogaty's professional portfolio built with React, Material UI, and Vite.

## Development Server

```bash
# Start the development server
pnpm dev
```

This will start a local development server at `http://localhost:5173`. The page will automatically reload when you make changes.

## Building

```bash
# Create a production build
pnpm build

# Preview the production build locally
pnpm preview
```

## Project Structure

```
jbcom.github.io/
├── src/              # React TypeScript source code
├── public/           # Static assets
├── content/          # Markdown content files
├── assets/           # CSS and images
├── docs/             # Documentation (you are here)
└── package.json      # Project configuration
```

## Available Scripts

| Command | Description |
|---------|-------------|
| `pnpm dev` | Start development server |
| `pnpm build` | Create production build |
| `pnpm preview` | Preview production build |
| `pnpm typecheck` | Run TypeScript type checking |
| `pnpm lint` | Run Biome linter |
| `pnpm lint:fix` | Auto-fix linting issues |
